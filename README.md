# Immunotherapy-Tumor-Suppression

This project contains the software intended for the 2021 IUSD District Science Showcase submission for Vishnu Manathattai and Pranav Sankar.
The tumor classification model identifies tumors from medical scans and images (MRI and CT).
The CAR-T cell program allows health providers to design immunotherapy processes from the input of a tumor. The program also allows custom antigens to be used to design CAR-T therapy.

FILE CONTENTS:
database.java contains a class and the main method to access the immunotherapy portal.
gene.java contains a class for the gene objects used in database.java.
tumorclassfication.zip contains the machine learning model for identifying tumors from medical imaging.
